# Multiscan Changelog

## [Initial Version] - 2026-03-09

- Paste any crypto address or transaction hash to auto-detect the chain
- Support for 70+ chains including Bitcoin, Ethereum, Solana, Cosmos ecosystem, and more
- ENS, SNS, and other name service resolution
- On-chain verification to confirm address activity
- Testnet detection and support
- Customizable block explorer preferences
- Chain logos for visual identification
